위치 기반

회식이 앱은 위치 정보를 기반으로 하고있습니다.

실시간 알림

간편함

Features기능 and Tech기술

회식이가 갖고있는 기능과 사용된 기술들에 대해서 설명되어있습니다.

계정 연동

회식이는 기본적으로 계정이 연동되어있습니다.

카카오 계정, 페이스북 계정, 일반 이메일로 로그인이 가능합니다.

앱 가이드

회식이는 앱을 처음 켰을 때 또는 보고 싶을 때 앱의 가이드를 확인할 수 있습니다.

  • 홈 화면
  • 신청서 화면
  • 견적서 화면
  • 계약서 화면

리뷰 시스템

회식이는 리뷰 시스템을 갖추고 있습니다.

고객이 식당에서 만족했던 점, 불편했던 점을 피드백 받을 수 있습니다.

식당에서는 리뷰를 이용한 홍보 효과를 얻을 수 있습니다.

다양한 사진

회식이에서는 식당의 다양한 사진을 확인할 수 있습니다.

  • 대표 사진
  • 매장 내부
  • 매장 외부
  • 메뉴판

자동 만료

불필요한 데이터를 가만히 냅두지 않습니다.

회식이는 시간이 지난 신청서를 자동으로 삭제해줍니다.

또한 연결되어있던 견적서도 함께 삭제해줍니다.

물론! 고객과 사장님 모두에게 알림을 보내줍니다.

페이징

회식이는 한번에 많은 양의 데이터를 가져오지 않습니다.

사용자가 필요한 만큼 일부씩 가져옵니다.

로컬 저장소

회식이는 중요한 데이터를 모두 보관해 놓고 있습니다.

네트워크 연결이 끊겨도 걱정하지 마세요.

  • 근처 식당
  • 근처 리뷰
  • 새로운 식당
  • 계약 내역

Trouble Shooting

소스코드 모듈화

2개 버전의 앱

2가지의 앱버전으로 인해 중복되는 코드가 많았습니다. 그래서 하나의 프로젝트에 모듈을 나눴습니다.

호환성

갤러리로부터 이미지 불러오기

갤러리로부터 이미지를 불러오는 과정에서 호환성 문제가 발생하였습니다. 여러 라이브러리로 대체해보았지만 원하는 방식이 없어 직접 코딩하였습니다.

MongoDB

처음으로 인한 지식의 부족함

MongoDB에서 거리를 계산하는데 쿼리 오류가 발생했습니다. MongoDB 공식 문서를 여러번 다시 읽고 데이터베이스를 처음부터 다시 구축하면서 해결하였습니다.

Crontab

메모리 해제 이슈 (Open - Close)

리눅스의 Crontab을 사용하는 과정에서 MongoDB를 터뜨렸습니다. Crontab의 로그를 확인하여 문제점 찾고 프로그램을 수정하여 문제를 해결하였습니다.

Review

부스트캠프 과정을 마친 후 제가 배운점, 느낀점들 입니다.

풍부해진 최신 기술 정보

부스트캠프에 입과하기 전에는 최신 기술 정보가 미약했지만, 현재는 다양한 정보를 보유하고 있습니다.

탄탄해진 CS 기초

알고있었지만 완벽하지 않았던 내용, 모르고 있었던 내용들을 부스트캠프 과정에서 다시 공부하는 계기가 되었습니다.

함께하는 즐거움

비록 부스트캠프 안드로이드 분야는 개인 프로젝트였지만 같은 공간에 개발 이야기를 나눌 수 있는 사람들이 함께 있다는 것이 즐거웠습니다.
