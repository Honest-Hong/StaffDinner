메시지 / 인원 수 / 시간 / 원하는 분위기 / 원하는 메뉴 / 위치
귀찮으면 매크로(▼)를 사용하세요!

실시간 알림

작성한 신청서에 사장님들께서 견적서를 보내주시면, 바로바로 알림이 도착합니다!

바로 견적서를 확인해서 식당들을 비교하고 원하는 식당에 계약을 하세요.
동영상
세부 기능

실시간 알람

새로운 신청서 알람, 새로운 견적서 알람, 계약 성공 알람, 계약 실패 알람이 실시간으로 전송됩니다.

위치 기반

모든 식당에게 알람을 보내지 않고 고객이 알리고 싶은 최대 거리 이내의 식당에게만 도착합니다.
모든 신청서의 내용을 보여주지 않고 식당과 가장 가까운 신청서 순서대로 보여줍니다.

자동 만료

신청서에 입력한 시간이 만료가 되면 자동으로 신청서와 견적서를 삭제해주며 취소 알람을 전송해줍니다.

페이징

내역을 불러올 때 모든 내역을 한번에 받아오지 않고 일부씩만 가져옵니다.

로컬 저장소

네트워크가 연결되지 않은 상태에서도 중요한 데이터를 확인할 수 있도록 로컬 저장소에 데이터가 저장되어있습니다.
